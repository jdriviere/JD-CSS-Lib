Buttons

Solid

Sparkle offers a variety of button colors, to go with your design needs.

Outline

If solid-colored buttons are not enough, Sparkle also offers the option of outlining the buttons. Simply add --outline at the end of the button type class, as in btn--primary--outline.

Ghost

Sparkle also offers the "ghost" button variety too. Simply add --ghost at the end of the button type class, as in btn--primary--ghost.

Disabled

You want to insert a button but not want it to be accessible yet? No problem! simply add btn--disabled to your list of classes, and use a code to remove that class whenever you need.

<!-- Button colors (solid) -->
<button class="btn btn--default btn--small">test</button>
<button class="btn btn--primary btn--small">test</button>
<button class="btn btn--success btn--small">test</button>
<button class="btn btn--warning btn--small">test</button>
<button class="btn btn--danger btn--small">test</button>
<button class="btn btn--light btn--small">test</button>
<button class="btn btn--dark btn--small">test</button>
<button class="btn btn--theme1 btn--small">test</button>
<button class="btn btn--theme2 btn--small">test</button>

<!-- Button colors (outline) -->
<button class="btn btn--default btn--small">test</button>
<button class="btn btn--primary btn--small">test</button>
<button class="btn btn--success btn--small">test</button>
<button class="btn btn--warning btn--small">test</button>
<button class="btn btn--danger btn--small">test</button>

<button class="btn btn--dark btn--small">test</button>
<button class="btn btn--theme1 btn--small">test</button>
<button class="btn btn--theme2 btn--small">test</button>

<!-- Button colors (disabled) -->
<button class="btn btn--default btn--small btn--disabled">test</button>

Sizes

Buttons come in different sizes for different purposes. You can alter a button size by adding btn--sizeName to the class list, where the sizeName is either one of the following: small, medium, large, or xlarge.





<!-- Button sizes -->
<button class="btn btn--theme1 btn--small">test</button>
<button class="btn btn--theme1 btn--medium">tes</button>
<button class="btn btn--theme1 btn--large">test</button>
<button class="btn btn--theme1 btn--xlarge">test</button>

Roundness

Sparkle offers the chance to even vary the roundness of its button design. Simply add btn--roundType in your class list, where roundType is one of the following: h-round (for 'half-round'), or f-round (for 'fully-round').

<!-- Button roundness -->
<button class="btn btn--theme1 btn--medium btn--h-round">test</button>
<button class="btn btn--theme1 btn--medium btn--f-round">test</button>

Block

You can also have your button take the entire span of the block too by adding btn--block to your class list. You can even mix it with the button roundness classes.

<!-- Button block -->
<button class="btn btn--theme1 btn--block">test</test>
<button class="btn btn--theme1 btn--block btn--h-round">test</test>
<button class="btn btn--theme1 btn--block btn--f-round">test</test>

Regular forms

Sparkle offers the possibility to customize forms. It was heavily influenced by Google Material UI. In order to use the forms feature, you must add the .form class to the the <form> element.
In order to optimize the use of <label> with its accompanying <input> tags, you must envelop them both in a <div> with a form__box class. And to stylize the inputs, simply add form__input to its class list (you do not need to add any classes to the <label> tag).

<form class="form">
<div class="form__box">
<input class="form__input" type="text" id="name">
<label for="input">Name</label>
</div>

<div class="form__box">
<input class="form__input" type="email" id="email">
<label for="input">Email</label>
</div>

<div class="form__box">
<input class="form__input" type="password" id="pass">
<label for="input">Password</label>
</div>
</form>
