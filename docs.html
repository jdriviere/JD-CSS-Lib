Alerts

Types

Alerts can serve to make sure users can see important messages on their screen. Simply create a block of code and stylize it accordingly. Start off by adding an alert, and then specify which type of alert you want by adding the second class, alert--$type, where $type is either one of the following: default, info, success, warning, or danger.

This is a default alert
This is an info alert
This is a success alert
This is a warning alert
This is a danger alert
Code:
<div class="alert alert--$type">
// Body goes here...
</div>
Components

Alerts also contain more features, such as a title, description, and list--if you wish to make your alerts cleaner.

  • Third alert in the list
Code:
<div class="alert alert--$type">
<h4 class="alert__title">Alert Title</h4>
<p class="alert__desc">Some text description here</p>
<ul>
<li class="alert__list">Some alert list #1</li>
<li class="alert__list">Some alert list #2</li>
<li class="alert__list">Some alert list #3</li>
</ul>
</div>

Buttons

Types (Solid):

Sparkle offers a variety of button colors, to go with your design needs. To specify the type of button you want, you must add btn--$type to your btn class, where $type is either one of the following: default, primary, secondary, success, warning, danger, light, or dark.

Code:
<button class="btn btn--$type">Some Text</button>
Types (Outlined):

If solid-colored buttons are not enough, Sparkle also offers the option of outlining the buttons. Simply add --outline at the end of the button type class, as in btn--$type--outline.

Code:
<button class="btn btn--$type--outline">Some Text</button>
Types (Ghosted):

Sparkle also offers the "ghost" button variety too. Simply add --ghost at the end of the button type class, as in btn--$type--ghost.

Code:
<button class="btn btn--$type--ghost">Some Text</button>
Sizes:

Buttons come in different sizes for different purposes. You can alter a button size by adding btn--$size to the class list, where the $size is either one of the following: small, medium, large, or xlarge.

Code:
<button class="btn btn--$type[--$option] btn--small">Some Text</button>
<button class="btn btn--$type[--$option] btn--medium">Some Text</button>
<button class="btn btn--$type[--$option] btn--large">Some Text</button>
<button class="btn btn--$type[--$option] btn--xlarge">Some Text</button>
Note:

Sparkle button features functions best when the button size is included in the button's class list.

Block:

You can also have your button take the entire span of the block too by adding btn--block to your class list. You can even mix it with the button roundness classes (see below).

Code:
<button class="btn btn--$type[--$option] btn--block">Some Text</button>
Roundness:

Sparkle offers the chance to even vary the roundness of its button design. Simply add btn--$roundness in your class list, where $roundness is one of the following: h-round (for 'half-round'), or f-round (for 'fully-round').

Code:
<button class="btn btn--$type[--$option] btn--h-round>Some Text</button>
<button class="btn btn--$type[--$option] btn--f-round>Some Text</button>
Disabled:

You want to insert a button but not want it to be accessible yet? No problem! simply add btn--disabled to your list of classes, and use a code to remove that class whenever you need.

Code:
<button class="btn btn--$type[--$option] btn--$size btn--disabled">Some Text</button>

Forms

Base

Sparkle offers the possibility to customize forms. It was heavily influenced by Google Material UI. In order to use the forms feature, you must add the .form class to the the <form> element.
In order to optimize the use of <label> with its accompanying <input> tags, you must envelop them both in a <div> with a form__box class. And to stylize the inputs, simply add form__input to its class list (you do not need to add any classes to the <label> tag).

Code:
<form class="form">
<div class="form__box">
<input class="form__input" type="text" id="input1">
<label for="input1">Input Text</label>
</div>

<div class="form__box">
<input class="form__input" type="text" id="input2">
<label for="input2">Input Text</label>
</div>

<div class="form__box">
<input class="form__input" type="password" id="input3">
<label for="input3">Input Text</label>
</div>
</form>
Note:

Make sure the <label> tag is placed underneath the <input> tag!

Tables

Base

In Sparkle, the table feature already comes fully responsive. There is no need to style each of its elements. Simple add the table class to the <table> tag.

ID First Name Last Name Age
1 John Smith 24
2 Sally Johnson 29
3 Rupert Flint 26
Code:
<table class="table">
<thead>
<tr>
<th>ID</th>
<th>First Name</th>
<th>Last Name</th>
<th>Age</th>
</tr>
</thead>

<tbody> <tr>
<td data-label="ID">1</td>
<td data-label="First Name">John</td>
<td data-label="Last Name">Smith</td>
<td data-label="Age">24</td>
</tr>

<tr>
