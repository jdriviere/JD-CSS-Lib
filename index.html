Welcome to the S3CSS Library

This is a small preview of my own personalized CSS framework called the S3CSS (Simple Stylesheet). Please, leave a comment in my Github if bugs are found, or you would like to make a suggestion(s). Also, please do not alter anything in the codes; and if you use it, please take the time to leave a positive (constructive) comment and credit. Thank you!

Why You Should Use It

Simple

I strived to make this framework as simple as it can get. No need for these pesky, hard to use frameworks!

Quick

Enough with these hard-coded frameworks that slow down your computer, through long, generated codes!

Free

Most importantly, it is FREE of charge! Nothing to pay. Ever! You can thank me later. ;)
